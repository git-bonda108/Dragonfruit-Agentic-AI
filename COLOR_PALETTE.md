# Dragonfruit MVP — Color Palette

Inspired by **[Dragonfruit Media](https://www.dragonfruitmedia.co/)** (#1 Global YouTube Agency): premium, clean, bold.

## CSS variables (in app)

| Variable | Value | Use |
|----------|-------|-----|
| `--df-bg` | `#FAFAFA` | Page background |
| `--df-bg-warm` | `#FFFBF7` | Warm tint option |
| `--df-surface` | `#FFFFFF` | Cards, sidebar |
| `--df-text` | `#1a1a1a` | Primary text |
| `--df-text-muted` | `#64748B` | Secondary text |
| `--df-accent` | `#E85D75` | Dragon fruit coral / CTA, selected tab, metric emphasis |
| `--df-accent-dark` | `#C94B62` | Hover state |
| `--df-success` | `#0d9488` | Connected / success |
| `--df-border` | `#E2E8F0` | Borders |
| `--df-shadow` | `0 1px 3px rgba(0,0,0,0.06)` | Subtle shadow |
| `--df-shadow-card` | `0 4px 20px rgba(0,0,0,0.06)` | Card shadow |

**Typography:** Plus Jakarta Sans (Google Fonts).

## Chart colors (Python `PALETTE` in mvp_app.py)
