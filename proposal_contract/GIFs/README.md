# Dragonfruit — Animated GIFs

All architecture and agent-flow animations live here. **Dragonfruit Media** colors and typography throughout.

---

## How to view

1. **Double-click** any `.html` file in this folder — it opens in your default browser and plays automatically.
2. Or **drag** the file into Chrome or Safari.
3. Or from Terminal: `open agentic_architecture_animated.html` (from inside the `GIFs` folder).

No server or login required. Animations loop indefinitely.

## How to record as a GIF

1. Open the HTML in your browser (as above).
2. Use a screen recorder that can export GIF:
   - **Mac:** QuickTime (record) + convert to GIF online, or **Kap** / **LICEcap**.
   - **Windows:** **ScreenToGif**, **LICEcap**.
   - **Online:** Record the tab, then use a “video to GIF” tool.
3. Record **1–2 loops** (about 8–12 seconds for the architecture; one full pass for the flow).
4. Export as GIF. Use Dragonfruit dimensions (e.g. 960×640 or your slide size).
