# Dragonfruit Media — Mock Data for Proposal & MVP

Generated from the provided pipeline, time allocations, tool stack, and AI survey. Use for demos, dashboards, and scenario planning.

## Files

- `pipeline_stages.json` — Pipeline phases and sample tasks with roles and tools
- `time_allocations.json` — Current vs dream % by role (from time audit)
- `tool_stack.json` — Tools, plans, monthly cost, who uses
- `pods_and_clients.json` — Sample pods, clients, and capacity
- `ai_survey_summary.json` — Role-level AI tool usage and top requests
- `wishlist_tools.json` — 16 wish list tools with priority and category
